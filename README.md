# Medical Insurance Cost Prediction

## Overview
This project predicts individual medical insurance costs using demographic and lifestyle factors such as age, BMI, and smoking status. It combines statistical modeling and machine learning to analyze key drivers of healthcare expenses.

## Key Insights
- Smoking is the strongest predictor of cost  
- BMI and age significantly increase charges  
- BMI × smoking interaction improves model performance  
- Nonlinear models outperform linear models in prediction  

## Methods
- Linear Regression (OLS)
- Interaction Models
- Random Forest
- Model diagnostics + bootstrap inference

## Results
- OLS: R² ≈ 0.75  
- Interaction Model: R² ≈ 0.84  
- Random Forest: R² ≈ 0.92
